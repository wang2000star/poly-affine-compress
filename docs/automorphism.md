# 布尔函数仿射约化：理论框架与算法设计

## 1. 问题定义

给定向量布尔函数 F: F₂ⁿ → F₂ᵏ，寻找仿射变换

```
z = Mx ⊕ b,    M ∈ F₂^{m×n}, b ∈ F₂^m
```

使得存在 G: F₂^m → F₂ᵏ，满足对任意 x ∈ F₂ⁿ：

```
G(Mx ⊕ b) = F(x)                    (1)
```

目标是极小化 G 的 ANF 单项式总数（或各分量 ANF 的并集规模）。

---

## 2. 与经典仿射等价的关系

### 2.1 经典定义

两个布尔函数 f, g: F₂ⁿ → F₂ 称为**仿射等价**，记作 f ≈ g，如果存在可逆仿射变换 A(x) = Mx ⊕ b（M ∈ GL(n, F₂)）使得：

```
f(x) = g(A(x))    ∀x
```

等价地，f 和 g 的 Walsh 谱在仿射变换下置换，代数次数保持不变。

### 2.2 我们的问题 vs 经典问题

| 维度 | 经典仿射等价 | 本问题 |
|------|------------|--------|
| 已知量 | f 和 g 都已知，判断是否等价 | 仅 F 已知，G 未知待求 |
| M 约束 | 可逆方阵 M ∈ GL(n) | 任意 M ∈ F₂^{m×n}，可非方、非满秩 |
| m 范围 | m = n（固定） | 0 ≤ m ≤ n（可变，可小于 n） |
| 目标 | 判定等价性 | 搜索使 T(G) 最小的 G |

**本质区别：** 经典仿射等价研究对称性（是否存在可逆变换连接两个已知函数），而我们的问题研究**约化性**（一个函数能否通过线性投影分解成更简单的函数）。

### 2.3 可逆情形的完全匹配

当 M 可逆时本问题退化为经典仿射等价：G(z) = F(M^{-1}(z ⊕ b))，即 G 就是 F 的仿射象。此时 T(G) 由 F 的 Walsh 谱排列决定，无法压缩。

**结论：** 有意义的压缩只可能来自：
1. **m < n**（投影压缩）
2. **m = n 但 rank(M) < n**（退化压缩，等价于 m < n）

## 3. 纤维丛理论

### 3.1 变换的几何结构

仿射变换 φ(x) = Mx ⊕ b 诱导 F₂ⁿ 上的分类：

```
x ∼ x'  ⇔  φ(x) = φ(x')
        ⇔  x ⊕ x' ∈ ker(M)
```

F₂ⁿ = ⋃_{z ∈ F₂^m} φ^{-1}(z)，每条纤维 φ^{-1}(z) = x₀ + ker(M)（对任意满足 φ(x₀) = z 的 x₀）。

**纤维数量 = 2^{rank(M)}**，每条纤维大小 = 2^{n - rank(M)}。

### 3.2 G 存在的充要条件

**定理 1（纤维常值条件）：** G 存在使得 (1) 成立，当且仅当 F 在每条纤维上为常数。即：

```
∀Δ ∈ ker(M), ∀x ∈ F₂ⁿ:  F(x ⊕ Δ) = F(x)
```

**证明：** 前文已给出。□

### 3.3 不变子空间

定义 F 的**平移不变子空间**：

```
Aut(F) = {Δ ∈ F₂ⁿ | ∀x: F(x ⊕ Δ) = F(x)}
```

**定理 1 的子空间表述：**

```
∃G: G∘φ = F  ⇔  ker(M) ⊆ Aut(F)
```

**性质 1：** Aut(F) ⊆ F₂ⁿ 是真子空间。
- 对向量布尔函数：Aut(F) = ⋂_{i=1}^k Aut(f_i)
- 对单输出 f：Aut(f) 的维数通过自相关函数确定

**性质 2（一致性单调性）：** 设 ker(M₁) ⊆ ker(M₂)（M₁ 纤维不粗于 M₂），则：
- M₂ 一致 ⇒ M₁ 一致
- M₁ 不一致 ⇒ M₂ 不一致

**推论：** 加行保持一致性，删行/换行可能破坏一致性。

### 3.4 维数下界

**定理 2（最小 m）：** 设 d = dim Aut(F)。任何满足 (1) 的变换必有：

```
m ≥ rank(M) ≥ n - d
```

**证明：** ker(M) ⊆ Aut(F) ⇒ dim ker(M) ≤ d ⇒ rank(M) = n - dim ker(M) ≥ n - d。而 m ≥ rank(M)。□

**达到下界的条件：** ker(M) = Aut(F) 且 m = n - d（M 的行空间恰是 Aut(F) 在 F₂ⁿ 中的补）。

### 3.5 商空间分解

**定理 3（正规形）：** 存在 G 使得 G∘φ = F 当且仅当 F 在商空间 F₂ⁿ/Aut(F) 上有定义。取 Aut(F) 的补空间 V，则限制 φ|_V 是单射，且：

```
F = G ∘ φ|_V
```

其中 G 定义在 dim V = n - d 维空间上。

**实际意义：** Aut(F) 的方向完全不包含信息。我们可以在 n' = n - d 维上搜索，而 Aut(F) 的 d 维方向可以直接消去。

## 4. ACCT 不变量

### 4.1 Walsh 谱

对单输出 f: F₂ⁿ → F₂，Walsh-Hadamard 变换：

```
Ŵ_f(α) = Σ_{x ∈ F₂ⁿ} (-1)^{f(x)} · (-1)^{α·x},    α ∈ F₂ⁿ
```

**性质（可逆变换下）：** 若 g(z) = f(M^{-1}(z ⊕ b)) 且 M 可逆，则：

```
Ŵ_g(α) = (-1)^{α·b} · Ŵ_f(M^T α)
```

即 Walsh 谱幅值在可逆仿射变换下**排列不变**。

**对我们的意义：** Ŵ_g 是 Ŵ_f 在行空间（激活的子空间）上的投影。当 m < n 时，Ŵ_g 的长度为 2^m ≈ 对应的 Ŵ_f 在变换后的限制。

### 4.2 自相关与 Aut(F)

自相关函数：

```
r_f(Δ) = (1/2ⁿ) · Σ_x (-1)^{f(x) ⊕ f(x⊕Δ)}
```

有 r_f(Δ) = 1 ⇔ Δ ∈ Aut(f)。通过 Wiener-Khinchin：

```
r_f = (1/2ⁿ) · W^{-1}(|Ŵ_f|²)
```

即先对 f 做 Walsh 变换，取模平方，再反变换即得所有 Δ 的 r_f 值。复杂度 O(n·2ⁿ)。

**对向量函数：** Aut(F) = ∩_i Aut(f_i)，逐分量取交。

### 4.3 代数次数

deg(f) = max{ |S| | S ⊆ [n], a_S ≠ 0 }，其中 a_S 是 ANF 系数。

**定理：** deg(G) = deg(F) 对任意变换 (M, b) 成立。

**证明：** 代入 z = Mx ⊕ b 到 G 的 ANF 中，不会产生高于 deg(G) 次的项；反之 G 由 F 通过解线性系统得到，不会降低次数。□

**实际意义：** 不必搜索 deg(G) < deg(F) 的 G——它们不存在。

### 4.4 依赖集（支持集）

对 f: F₂ⁿ → F₂ 和变量 x_j：

```
x_j 影响 f  ⇔  ∃x: f(x) ≠ f(x ⊕ e_j)
              ⇔  e_j ∉ Aut(f)
```

**定义依赖图：** D(F) = {j | e_j ∉ Aut(F)}。即所有被 F 实际使用的输入变量。

**依赖筛选：** 池子行 w 只在 supp(w) ∩ D(F) ≠ ∅ 时才有可能有效。否则 w 筛分的是 F 不依赖的变量。

## 5. Aut(F) 的计算

### 5.1 Truth Table 方法（n ≤ 25）

对 n ≤ 25，2ⁿ 个点的 truth table 可以存放（最大 4GB for n=25）。

**Walsh 功率谱法（n ≤ 20，推荐）：**

```
1. 对每个输出 f_i:
   a. Ŵ = Walsh-Hadamard(f_i)       O(n·2ⁿ)
   b. S[α] = Ŵ[α]²                   O(2ⁿ)
   c. r = Inverse_Walsh(S)           O(n·2ⁿ)
   d. Aut(f_i) = {Δ | r[Δ] = 1}      O(2ⁿ)
2. Aut(F) = ∩_i Aut(f_i)
```

**直接平移法（n ≤ 16，替代）：**

对每个非零 Δ，检查 f_i 是否在 Δ 方向常数。用 bit 操作扫描 whole truth table：

```python
for Δ in 1..2ⁿ⁻¹:
    for w in 0..n_words-1:
        if f_tt[w] != xor_shift(f_tt, Δ, w):  # 平移 Δ 后的 word
            break  # Δ ∉ Aut(f)
    else:
        Aut(f).append(Δ)
```

其中 `xor_shift(f_tt, Δ, w)` 返回 f_tt 平移 Δ 后的第 w 个 word。借助 `vpternlog`/`_mm512_xor_si512` 等 SIMD 指令，64 位同时处理。

### 5.2 SAT 方法（n ≥ 26）

对每个输出 f_i 和候选 Δ，判断 Δ ∉ Aut(f_i) 等价于求解：

```
SAT(∃x: f_i(x) ≠ f_i(x ⊕ Δ))
```

**编码方案 A：Δ 作为自由变量（一次性检测所有 Δ）**

```
Input variables:  x_0...x_{n-1}, Δ_0...Δ_{n-1}, internal_wires_1, internal_wires_2
Constraints:
  (1) Δ ≠ 0                   (至少一个 Δ_i = 1)
  (2) x'_i = x_i ⊕ Δ_i        (每个 i, XOR 门)
  (3) circuit(x) → y          (电路副本 1)
  (4) circuit(x') → y'        (电路副本 2)
  (5) y ⊕ y' = 1              (输出不同)
```

**优化：** 用 CryptoMiniSat 原生 XOR 约束处理 (2)。对 n=32, 100+ 门的电路，通常毫秒级求解。

**编码方案 B：固定 Δ（多次调用，用于渐进 M 构建）**

每次只检查特定 Δ（由排他性线性方程解出）。编码简化为：

```
Input variables:  x_0...x_{n-1}
Constraints:
  (1) x'_i = x_i ⊕ Δ_i        (Δ_i 已知常数, NOT 门或恒等)
  (2) circuit(x) → y
  (3) circuit(x') → y'
  (4) y ⊕ y' = 1
```

Δ_i = 0 时 x'_i = x_i（省略门），Δ_i = 1 时 x'_i = ¬x_i（NOT 门）。变量数减半，求解更快。

### 5.3 采样法（快速近似）

```
function sample_check(f, Δ, N=2000):
    for t = 1..N:
        x ← random F₂ⁿ
        if f(x) ≠ f(x ⊕ Δ): return True   # Δ 影响了 f
    return False                            # 未检测到影响
```

可靠性：若 f 在 Δ 方向实际改变的比例为 p，N 次采样至少发现一次的概率为 1 - (1-p)^N。对 p ≥ 0.001，N=3000 达 95% 置信度。

### 5.4 各规模方法汇总

| n | Truth Table | Aut(F) | 依赖集 | 推荐方法 |
|---|------------|--------|--------|---------|
| ≤ 16 | 2ⁿ ≤ 64KB | Walsh 功率谱 | TT 精确扫描 | Walsh + TT |
| 17-20 | ≤ 1MB | Walsh 功率谱 | TT 精确扫描 | Walsh + TT |
| 21-25 | ≤ 512MB | Walsh (浮点) | 采样 | Walsh + 采样 |
| 26-32 | 电路求值 | SAT 方案 A | 采样 | SAT |
| 33+ | 电路求值 | SAT 方案 A | 采样 | SAT |

## 6. 基于 Aut(F) 的搜索算法

### 6.1 搜索管线

```
Phase 0: 输入 F, 构建表示 (TT / FastCircuit / SAT formula)
Phase 1: 计算 Aut(F) (dim d, 一组基)
Phase 2: 计算依赖集 D(F)
Phase 3: 构建候选池 (3n-row pool), 按 Aut(F) + D(F) 筛选
Phase 4: 渐进 M 构建:
  4a. 起始: M_curr = 从 identity 的 n 行 (ker={0}, 始终一致)
  4b. 候选: 依次尝试 pool 中的行 w (按 Walsh 评分排序)
  4c. 排他: 解 M_curr·Δ=0, w·Δ=1 → 无解则跳过
  4d. 有效: Δ ∈ Aut(F)? → 跳过 (划分无效)
  4e. 接受: M_curr ∪= [w], 更新 ker(M_curr) 基
  4f. 直到 |M_curr| = max_m 或 ker(M_curr) = Aut(F)
Phase 5: 评估: 计算 G 的 ANF (n≤20 TT / n≥21 z-space)
Phase 6: 验证: 随机点 / 穷举 (n≤16)
```

### 6.2 依赖集筛选

```
对每个输出 f_i:
  D_i = {j | e_j ∉ Aut(f_i)}

池子行 w 的支撑集 supp(w) = {位置 w 有 1 的 bit index}。

行 w 对输出 f_i 有效的必要条件:
  supp(w) ∩ D_i ≠ ∅

联合变换 (策略 1):
  D(F) = ⋃_i D_i
  行 w 有效的必要条件: supp(w) ∩ D(F) ≠ ∅
```

### 6.3 Aut(F) 正交筛选

```
行 w 绝对无效的条件 (对任意 M_curr):
  w ∈ Aut(F)⊥ = {v | v·Δ = 0 ∀Δ ∈ Aut(F)}

证明: 若 w ∈ Aut(F)⊥, 则对任意 Δ 有 w·Δ = 0 或 Δ ∉ Aut(F)。
因此不存在 Δ 满足 w·Δ = 1 且 Δ ∈ Aut(F)。
由排他性检验 (§4.4), w 不可能为有效行。□
```

### 6.4 渐进 M 构建的排他性

```
给定 M_curr (r 行), 候选行 w。

Step 1: 解线性方程 M_curr·Δ_proj = 0 且 w·Δ_proj ≠ 0
  - 即: w 的行向量不在 span(M_curr) 中
  - 等价于 rank(M_curr ∪ [w]) = r + 1
  - 简单行阶梯检验即可

Step 2: 在 ker(M_curr) 中找 Δ 使 w·Δ = 1
  - 高斯消元解 [M_curr; w]·Δ = [0; 1]
  - 无解 → w 冗余, 跳过

Step 3: 检查 Δ ∈ Aut(F)? 
  - 是 → w 划分了无关方向, 跳过
  - 否 → w 有效, 加入 M_curr

Step 4: 更新 ker(M_curr)
  - 新 ker = 旧 ker ∩ ker(w)
  - 基更新: 旧基中满足 w·v = 0 的保留, 不足 dim-1 时从 w·v ≠ 0 的基组合构造
```

### 6.5 与古典仿射搜索的互操作

**Walsh 方向搜索的再解释：**

Walsh 谱幅值 |Ŵ(α)| 衡量的是 f(x) 与 x·α 之间的相关性。高幅值 |α| 对应的方向 α 是"最接近线性"的方向。

在渐进 M 构建的上下文中，Walsh 评分高的方向 α 倾向于：

```
∃x: f(x) ≠ f(x ⊕ α)     (高概率)
```

因为高 Walsh 幅值意味着 f 在该方向有显著的变化。这与我们的排他性检验直接互补：

```
排他性检验: 解 Δ (w 方向) 再检查 Δ ∈ Aut(F)?
Walsh 筛选:  直接用 Walsh 幅值估计 w 的有效概率
```

### 6.6 策略 1（联合）vs 策略 2（分量独立）

| 策略 | Aut(F) | m 范围 | 优势 | 劣势 |
|------|--------|--------|------|------|
| 1（联合） | ∩Aut(f_i)，更小 | m ≥ n - dim ∩Aut(f_i) | 最终 z 空间小，搜索次数少 | 每个输出受限 |
| 2（独立） | Aut(f_i) 各自，更大 | m_i ≥ n - dim Aut(f_i) 各自 | 每个输出最优 | 合并后 z 空间大 |

**混合策略（推荐）：**

1. 对每个输出 f_i 计算独立 M_i（策略 2）
2. 合并各 M_i 的行构造联合 M
3. 若合并后 m 太大，用贪婪法选最优子集

## 7. 已知实例的再分析

### 7.1 hd08 (n=8, k=1)

- 单输出，Aut(F) 可能较小
- 已知最优 m=6（Union T=6），压缩率 6/8
- 解释：rank(M) = 6，ker(M) 为 2 维
- 需确认是否 ker(M) ⊆ Aut(F)

### 7.2 hd01 (n=32, k=32)

- 每个输出 f_i 依赖极少数输入（部分输出 T=1）
- 对输出 f_i，Aut(f_i) 包含所有与 f_i 无关的 e_j
- dim Aut(f_i) 大 → m_i 下界小
- 排列 M 正是利用了 Aut(f_i) 的高度结构

### 7.3 hd10 (n=32, k=32, 仅 2 非平凡输出)

- 大部分输出为常数（T=0），Aut(f_i) = F₂³²
- 非平凡输出依赖于少量输入
- 搜索可聚焦于少数输出的依赖集

## 8. 开放问题

1. **最优 M 的存在性：** 对任意 F，是否存在 M 使 T(G) 达到全局最小？还是只能搜索局部最优？

2. **Ker(Aut(F)) 的结构：** Aut(F) 的补空间 V 的选取是否影响极小 T(G)？即取不同的补空间会得到不同的 G？

3. **Walsh 引导的理论保证：** 高 Walsh 幅值方向是渐进最优选行的理论依据？是否有反例？

4. **SAT 的可扩展边界：** 对 n ≥ 40, k ≥ 64 的大型电路，SAT 求解 Aut(F) 的可行性？

5. **近似约化：** 若放弃精确等式 G(Mx⊕b) = F(x)，允许 ε 误差，压缩率能提升多少？（与 LRC、AE 等编码理论相关）

6. **非满秩 M 的纤维结构：** rank(M) < m 时，部分 z 是冗余的。能否在渐进构建中自动剪枝这些冗余维度？
