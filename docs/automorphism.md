# 布尔函数仿射约化：纤维丛理论与自动机方法

## 1. 问题重述

给定向量布尔函数 F: F₂ⁿ → F₂ᵏ，寻找仿射变换

```
z = Mx ⊕ b,    M ∈ F₂^{m×n}, b ∈ F₂^m
```

使得存在 G: F₂^m → F₂ᵏ，满足对任意 x ∈ F₂ⁿ：

```
G(Mx ⊕ b) = F(x)
```

目标是**极小化 G 的 ANF 总单项式数**（或各分量 ANF 的并集的规模）。

---

## 2. 纤维丛理论

### 2.1 变换的几何图景

仿射变换 φ(x) = Mx ⊕ b 诱导 F₂ⁿ 上的一个等价关系：

```
x ∼ x'  ⇔  φ(x) = φ(x')
                ⇔  M(x ⊕ x') = 0
                ⇔  x ⊕ x' ∈ ker(M)
```

F₂ⁿ 被划分成 2^m 条纤维，每条纤维是 ker(M) 的陪集：

```
F₂ⁿ ≅ ⋃_{z ∈ F₂^m} φ^{-1}(z)
```

每条纤维包含 2^{n-m} 个点。

### 2.2 G 存在的充要条件

**定理 1（一致性）：** G 存在当且仅当 F 在每条纤维上为常数。即：

```
∀Δ ∈ ker(M), ∀x ∈ F₂ⁿ: F(x ⊕ Δ) = F(x)
```

**证明：**

(⇒) 若 G 存在，取 Δ ∈ ker(M)。任给 x：

```
F(x ⊕ Δ) = G(M(x ⊕ Δ) ⊕ b) = G(Mx ⊕ MΔ ⊕ b) = G(Mx ⊕ b) = F(x)
```

(⇐) 若 F 在每条纤维上常数，定义 G(z) = F(x) 对任意 x ∈ φ^{-1}(z)。由纤维常数的条件，G 良定义。此外由构造有 G∘φ = F。□

### 2.3 不变子空间

定义 F 的**不变子空间**（Invaraint Subspace）：

```
Aut(F) = {Δ ∈ F₂ⁿ | ∀x ∈ F₂ⁿ: F(x ⊕ Δ) = F(x)}
```

即那些使 F 整体平移不变的向量全体。

**性质：**

1. **Aut(F) 是 F₂ⁿ 的子空间。**
   - 0 ∈ Aut(F) 平凡成立
   - 若 Δ₁, Δ₂ ∈ Aut(F)，则 (Δ₁ ⊕ Δ₂) ∈ Aut(F)
   - 证：F(x ⊕ Δ₁ ⊕ Δ₂) = F((x ⊕ Δ₁) ⊕ Δ₂) = F(x ⊕ Δ₁) = F(x)

2. **Aut(F) 是各输出分量 Aut(f_i) 的交：**

   ```
   Aut(F) = ⋂_{i=1}^k Aut(f_i)
   ```

3. **一致性条件的子空间表述：**

   ```
   G 存在  ⇔  ker(M) ⊆ Aut(F)
   ```

### 2.4 理论下界

**定理 2（最小压缩维数）：**

```
m ≥ n - dim Aut(F)
```

**证明：** 由 ker(M) ⊆ Aut(F) 得 rank(M) ≥ n - dim Aut(F)。而 m ≥ rank(M)，所以 m ≥ n - dim Aut(F)。□

**推论：** m_min = n - dim Aut(F) 是理论最优——不可能用少于 m_min 个 z 变量一致表示 F。达到此下界当且仅当 ker(M) = Aut(F)，即 M 的零空间恰好等于 Aut(F)。

### 2.5 结构定理：分解与自然表示

**定理 3（商空间表示）：** 存在 G 当且仅当 F 可以分解为复合映射：

```
        φ        ˜F
F₂ⁿ ────→ F₂^m ──→ F₂ᵏ
   ↘              ↗
      F  (fiber-constant)
```

自然表示发生在商空间 F₂ⁿ/Aut(F) 上。令：

- n' = n - dim Aut(F)
- 取 Aut(F) 的补空间 V（任意），dim V = n'
- 限制 φ 到 V 得到同构：V ≅ F₂^{n'} → F₂^m

此时 G 定义在子空间 V 上，压缩率为 n'/n。

## 3. Aut(F) 的算法

### 3.1 向量布尔函数的自相关

对单输出 f: 定义自相关函数：

```
r_f(Δ) = (1/2ⁿ) Σ_{x ∈ F₂ⁿ} (-1)^{f(x)} · (-1)^{f(x⊕Δ)}
         = (1/2ⁿ) Σ_x (-1)^{f(x) ⊕ f(x⊕Δ)}
```

**性质：**
- r_f(0) = 1
- r_f(Δ) = 1 ⇔ Δ ∈ Aut(f)
- r_f(Δ) ∈ [-1, 1]，值越大表示 Δ 方向的不变性越强

通过 **Wiener-Khinchin 定理**：

```
Ŵ_f(α) = Σ_x (-1)^{f(x)} · (-1)^{α·x}    (Walsh-Hadamard 变换)
r_f(Δ) = (1/2ⁿ) · Σ_α |Ŵ_f(α)|² · (-1)^{α·Δ}     (功率谱反变换)
```

所以计算流程：

```
W = Walsh-Hadamard(f)        O(n·2ⁿ)
S[α] = |W[α]|²               O(2ⁿ)
r = Inverse_Walsh(S)          O(n·2ⁿ)
Aut(f) = {Δ | r[Δ] = 1}      O(2ⁿ)
```

**数值容差：** 实际计算中 r[Δ] 可能接近 1 但不是精确 1（浮点误差）。取阈值 r[Δ] > 1 - ε。

### 3.2 向量函数的 Aut(F)

```
Aut(F) = ⋂_{i=1}^k Aut(f_i)
```

实现 = 各输出自相关结果取 AND。

### 3.3 各规模适配

| n 范围 | 方法 | 复杂度 | 确定性 |
|--------|------|--------|--------|
| n ≤ 20 | Truth table + Walsh 功率谱 | O(k·n·2ⁿ) | 精确 |
| 21 ≤ n ≤ 25 | Truth table + Walsh | O(k·n·2ⁿ) | 精确 |
| n ≥ 26 | SAT 逐 Δ 检测 或 随机采样 | 见 §3.4 | 精确/概率 |

### 3.4 SAT 编码

对于用电路给出的 F（n ≥ 26），用 SAT 逐 Δ 检测 **Δ ∉ Aut(f)**：

**命题：** Δ ∉ Aut(f) ⇔ 存在 x 使得 f(x) ≠ f(x⊕Δ)。

SAT 编码：对每个输出分量 f_i，构建两个电路的 XOR：

```
// 电路副本 1: 输入 x，输出 y
circuit(x, w1, ..., w_r, y)
// 电路副本 2: 输入 x' = x ⊕ Δ，输出 y'
circuit(x ⊕ Δ, w1', ..., w_r', y')

assert: (y ⊕ y') = 1
```

**关键编码细节：**

#### Δ 的编码方式

有两种方式：

**方法 A：Δ 作为自由变量。** 一次 SAT 求解直接得到反例 (x, Δ)。

```
∃ x_0..x_{n-1}, Δ_0..Δ_{n-1}:
  (Δ ≠ 0)
  ∧ [x_i' = x_i ⊕ Δ_i]
  ∧ [circuit(x) ⊕ circuit(x') = 1]
```

同时求解 x 和 Δ 的好处：一次调用即可证 Δ 存在与否。坏处：Δ 的自由度引入更多变量。

**方法 B：固定 Δ 再解。** 对每个候选 Δ，SAT 只判断存在性。

```
// 固定 Δ 后，电路 1 和 2 的输入变量是 x 和 x⊕Δ
// 编码为 x' = x ⊕ Δ（Δ 固定）
// 公式简化为：∃ x: circuit(x) ⊕ circuit(x⊕Δ) = 1

// 等价编码：不引入 x' 变量，在电路中直接替换
// x_i' = (x_i ⊕ Δ_i)  = x_i when Δ_i=0, ¬x_i when Δ_i=1
// 用 ITE 门实现：x_i' = ITE(Δ_i, ¬x_i, x_i)
```

当候选 Δ 数量少时 B 更优。但如果要穷举检测 Aut(F)，A 更高效。

**方法 A 的 CNF 优化技巧：**

对 Δ 的使用可以优化——不是所有 Δ 变量都参与。线性约束 `M_curr · Δ = 0` 和 `w·Δ = 1` 可以将 Δ 的部分维度表示为其他维度的线性组合，减少 SAT 变量数：

```
// 对 (n - rank(M_curr)) 个自由 Δ 变量进行 SAT 搜索
// 其余的 Δ_j 由线性方程组确定
```

#### XOR 链的 CNF 编码

对于 AND/XOR 形式的电路（如 hd01/hd02/hd10），两个实例的 XOR 输出可以用标准 Tseitin 变换：

```
// 门的编码：
// z = a ⊕ b:  (¬a ∨ ¬b ∨ ¬z) ∧ (a ∨ b ∨ ¬z) ∧ (¬a ∨ b ∨ z) ∧ (a ∨ ¬b ∨ z)
//   = a ∧ b:  (¬a ∨ ¬b ∨ z) ∧ (a ∨ ¬z) ∧ (b ∨ ¬z)
//   = a ∨ b:  (a ∨ b ∨ ¬z) ∧ (¬a ∨ z) ∧ (¬b ∨ z)

// x' = x ⊕ Δ:
// 对每个 Δ_i, 如果 Δ_i 是自由变量:
//   x'_i = x_i ⊕ Δ_i
// 编码: x'_i 是 x_i 和 Δ_i 的 XOR
// 但如果 Δ_i 固定为 0: x'_i = x_i（省略 XOR 门）
// 如果 Δ_i 固定为 1: x'_i = ¬x_i（一个 NOT 门）
```

#### 增量式 SAT

当渐进构建 M 时，每次加入新行只需**增量式**更新 SAT 约束：

```
// 已有 M_curr (r 行) 的约束已经求解完毕
// 添加新行 w:
//  新增约束: w·Δ = 1
//  提交到 SAT 求解器（增量模式，保留之前的学习子句）
```

MiniSat / Glucose / CryptoMiniSat 都支持增量求解。

#### SAT 求解器的选择

| 求解器 | 特点 | 适用场景 |
|--------|------|----------|
| MiniSat | 轻量，C++ 嵌入方便 | 小规模、少调用 |
| CryptoMiniSat | 原生 XOR 支持 | XOR 密集型电路、大量 SAT 调用 |
| CaDiCaL | 含 IPASIR 增量接口 | 大量增量调用 |
| Kissat | 最快之一 | 单次复杂查询 |

对于 hd01/hd02/hd10（121-231 门），CryptoMiniSat 最有优势，因为基准电路大量含 XOR。

#### 多次调用的缓存

对每个输出的 Aut(f_i) 检测，候选 Δ 的检查结果可以缓存：

```
cache[Δ] = {SAT(∃x: f_i(x) ≠ f_i(x⊕Δ)) ? active : inactive}
```

渐进 M 构建中，ker(M_curr) 改变时，只需要重新检查**不在当前 ker(M_curr) 内**的那些 Δ。

### 3.5 采样法（快速近似）

当不需要完全精确的 Aut(F) 时：

```
Δ_active(Δ): 对 t=1..N:
  x ← random
  用 FastCircuit 求值 f(x) 和 f(x⊕Δ)
  if f(x) ≠ f(x⊕Δ): return True
return False
```

**N 的选择：** 若 f 在 Δ 方向改变的比例为 p，则 N 次采样至少发现一次的概率为 1 - (1-p)^N。对于 p ≥ 0.001（千分之一），N = 3000 给出 95% 置信度。

## 4. 基于 Aut(F) 的搜索算法

### 4.0 一致性的单调性

**定理 4（单调性）：** 设 M 和 M' 是两个变换矩阵，ker(M) ⊆ ker(M')（即 M 的纤维不粗于 M'）。则：

```
(1) 若 M 一致（∃G: G(Mx⊕b) = F(x)），则 M' 也一致
(2) 若 M' 不一致，则 M 也不一致
```

**证明：** 一致性等价于 ker(M) ⊆ Aut(F)。由 ker(M) ⊆ ker(M')：
- M 一致 ⇒ ker(M) ⊆ Aut(F)。而 ker(M') ⊆ ker(M)，故 ker(M') ⊆ Aut(F) ⇒ M' 一致。
- (2) 是 (1) 的逆否。□

**推论 1（子矩阵）：** 若 (M,b) 不一致，则它的任意子矩阵（删行）也不一致。
**推论 2（扩充）：** 若 (M,b) 一致，则它的任意扩充矩阵（加行）也一致。

**实际意义：**

```
identity (m=n, ker={0})
  ├── 一致（平凡，∀F）
  ├── 加行 → 仍一致（核更小）
  ├── 换行（保持行空间）→ 仍一致（核不变）
  └── 换行（改变行空间）→ 可能不一致（核变大）
```

这给出了渐进构建的安全策略：

```
Step 1: 从 identity 出发 (m=n)       ← 始终一致
Step 2: 加 complement 行 (m>n)       ← ker不变, 仍一致
Step 3: 替换/交换行                  ← 可能不一致! 需检测
```

即：**加行安全，删行/换行危险。** 危险操作后需要重新验证一致性。

### 4.1 搜索空间降维

给定 Aut(F)（维度 d）:

1. **计算基底：** 选 Aut(F) 的一组基 {a₁, ..., a_d}
2. **补空间：** 求直和补 V，dim V = n' = n - d
3. **在新的 n' 维空间上搜索**

实际操作：构造变换矩阵 T 使得 Aut(F) 的基向量被映射到后 d 个坐标。搜索时只在前 n' 维做文章。

### 4.2 渐进式 M 构建

```
输入: F, pool_rows, max_m
初始化: M = []  (0 行)
        V_null = {基向量集合: 对偶基 of F₂ⁿ}  (当前 ker(M) 的一组基)

while |M| < max_m:
  找到最佳 w ∈ pool_rows:
    score = 0
    for Δ ∈ basis(V_null):
      if w·Δ = 1:
        if not SAT_exists_x(F(x) ≠ F(x⊕Δ)):
          score -= 1  // w 的信息被 F 的不变性抵消
        else:
          score += 1  // w 划分了新的有效信息
  
  选 score 最大的 w
  M = M ∪ [w]
  更新 V_null = ker(M) 的一组基
  if V_null == {0}: break  // ker(M) 已小到 trivial
```

### 4.3 行筛选（预筛）

在 pool 构建阶段，对每行 w：

```
// 行 w 绝对无效的判定
// 令 supp(w) = {i₁,...,iₖ}

// 条件 1: 完全不等于任意 Δ ∈ Aut(F)
// 即：∀Δ: w·Δ = 0 对所有 Δ ∈ Aut(F) 成立?
// 等价于 w ∈ Aut(F)⊥（正交补）

if w ∈ Aut(F)⊥: 
  跳过  // w 正交于所有不变方向，不可能表达新信息

// 条件 2（更精细）：支撑集完全不在依赖集中
dep_set = 输出分量的依赖集（实际受影响的输入 bit）
if supp(w) ∩ dep_set = ∅:
  跳过  // w 不作用于任何相关变量
```

### 4.4 渐进选行中的排他性检测

```
bool row_is_redundant(M_curr, w):
  // 求解 M_curr · Δ = 0 且 w · Δ = 1
  // 构造增广矩阵 [M_curr | 0; w | 1]
  // 高斯消元判断是否有解
  
  if 无解: return True  // w 的信息完全被 M_curr 覆盖
  if 有解 Δ:
    // 检查是否 Δ 方向 F 不变
    if Δ ∈ Aut(F): return True  // 无效划分
    return False  // w 真正有效
```

### 4.5 核子空间的增广更新

渐进选行中，每增加一行 w，ker(M) 缩小：

```
旧 ker(M) = {Δ | M·Δ = 0}
新 ker(M') = {Δ | M·Δ = 0 ∧ w·Δ = 0}
               = ker(M) ∩ ker(w)
```

ker(M) 的基更新：
```
// 对旧基 {b₁, ..., b_{n-r}}
// 新基是其中满足 w·b_j = 0 的子集
// 加上新有效 Δ 之间的线性组合
```

具体算法：

```
输入: old_basis (ker(M) 的基), w
输出: new_basis (ker(M') 的基)

new_basis = []
for v in old_basis:
    if w·v == 0:
        new_basis.append(v)

// 当 old_basis 中所有 w·v=1 时，需要找组合
// 如果 dim ker(M) = s，w 的约束可能只消去一个维度
// 需要处理：在 ker(M) 中找到 w·v=1 的 v，再与其他组合获取 w·v=0 的新基
// 标准做法：在 ker(M) 中高斯消元加入 w 行
```

## 5. 完整管线设计

### 5.1 管线总览

```
┌─────────────────────────────────────────────┐
│ Phase 0: 输入电路 F: F₂ⁿ → F₂ᵏ              │
│          构建 FastCircuit / TruthTable       │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│ Phase 1: Aut(F) 计算                        │
│  n≤20: Walsh 功率谱 + 反变换（O(n·2ⁿ)）     │
│  n≥21: SAT 逐 Δ 检测                        │
│  输出: dim Aut(F), 一组基 {a₁,...,a_d}       │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│ Phase 2: 依赖集分析                          │
│  对每个输出 f_i:                            │
│    哪些 x_j 影响 f_i?                       │
│  n≤20: truth table 精确（平移 e_j 检测）     │
│  n≥21: SAT 或采样                           │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│ Phase 3: 构建候选池 (3n-row pool)            │
│  - 过滤: 支撑集与依赖集无关 → 跳过          │
│  - 过滤: w ∈ Aut(F)⊥ → 跳过                 │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│ Phase 4: 渐进 M 构建（每输出独立 / 联合）    │
│  - 选 Walsh 评分最高的 w                     │
│  - 排他性检测: 解 M_curr·Δ=0, w·Δ=1         │
│  - Δ ∈ Aut(F)? → 跳过                       │
│  - 否则加入 M                               │
│  - 更新 ker(M) 的基                         │
│  - 直到 |M| = max_m 或 ker(M) = {0}         │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│ Phase 5: g(z) ANF 最终评估 + 验证           │
│  n≤20: truth table 精确计算                 │
│  n≥21: z-space 遍历+电路求值                │
└─────────────────────────────────────────────┘
```

### 5.2 策略 1（共享 M）vs 策略 2（分量独立 M）

**策略 1（联合）：** 所有输出共享同一个 M 和 Aut(F)

```
Aut(F) = ⋂_i Aut(f_i)        ← 更小 → ker(M) 更受限 → m 更大
```

**策略 2（分量独立）：** 每个输出 f_i 有自己的 M_i

```
Aut(f_i) 各自独立            ← 更大 → ker(M_i) 更大 → m_i 更小
但合并后的 z 空间是各 m_i 之和
```

### 5.3 已知问题的再解释

**hd01 (n=32, k=32):** 每个输出 T=1（常数或单个变量）
- Aut(f_i) 是大子空间（因为 f_i 只依赖一个 x_j）
- 补空间 V 维度很低
- Python opt2 找的排列 M 恰好利用了高度结构化的不变子空间

**hd08 (n=8, k=1):** 单输出
- Aut(F) 可能为 {0}（无平移不变性）
- 但存在压缩（m=8 → 6），通过非满秩 M 实现
- 说明最压缩不一定需要 ker(M) ⊆ Aut(F)——非满秩 M 可以在信息损失条件下实现更小的 m

**关键区别：** 非满秩 M（m < n）允许信息损失（部分纤维内 F 不是常数），但这样的 G 只在近似意义下成立。我们要求精确等式 G(Mx⊕b) = F(x)，所以一致性条件是必须的。

## 6. 开放问题

1. **非满秩 M 的精确等式：** 当 m < n 且 rank(M) < m 时，纤维的划分不是 ker(M) 的陪集。此时存在 g 的条件是否更弱？

   **回答：** 不。对任意 M，纤维的划分始终是 ker(M) 的陪集。rank(M) = r ≤ m，纤维数量为 2ʳ ≤ 2^m。多余的 m-r 个 z 维度是冗余的。一致性条件仍然是 ker(M) ⊆ Aut(F)。

2. **m 的搜索范围：** 理论上界 m ≥ n - dim Aut(F)，在何时可达？

3. **与 Walsh 搜索的关系：** Walsh 谱幅值高的方向是否对应 Aut(F) 外但接近 Aut(F) 的方向？

4. **SAT 求解的可扩展性：** 对于 n=32, k=32, 100+ 门的电路，穷举 Aut(F) 的 SAT 求解需要多少步（最多 2³² 个 Δ 方向需要逐 Δ 检查）？
